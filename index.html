Can't connect to game server

Possible reasons and fixes:

  • Server not running: Start the game server on the machine you entered (run the Node server or use a tunnel).
  • Wrong host or port: Ensure you entered the correct host:port (or full ws://host:port).
  • Firewall or network: The host machine may block incoming connections — try connecting from the same machine first.
  • Using file://: If opened from file, you must provide a reachable WebSocket server address when prompted.
  • HTTPS page: If the page is served with HTTPS, use a wss:// server (secure WebSocket).

Quick steps: start the server, or run ngrok http 3000 and use its host, or host the `my_game` folder on a server.
